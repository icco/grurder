# grurder

This is a norns script. You can learn more about Norns at https://monome.org/docs/norns/scripting/. This script uses a Midi in from 8mu (https://www.musicthing.co.uk/8mu_page/) and outputs midi to mmMidi
(https://busycircuits.com/pages/alm023). The input from 8mu should be arguments to a generative sequencer which changes how it iterates over time. The sequence generated for two sets of midi which is sent to
mmMidi should be visualized on screen. There should also be a seperate screen that shows what you are getting from 8mu.

For iudeas on generating sequences, take a look at https://en.wikipedia.org/wiki/Music_and_artificial_intelligence and https://www.reddit.com/r/synthesizers/comments/1rqw1ix/call_for_generative_music_algorithms/
 and https://www.musicthing.co.uk/books.html.
